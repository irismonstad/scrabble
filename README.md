# Simplified scrabble

## "Roadmap"
### v1:
* Get randomised letters from "bag"
* Place letters on field

### v2:
* Check validity of word (dictionary API)

### v3:
* Add points to letters
* Score calculation

### v4:
* Special fields on board (e.g. double letter, triple word)

## Functionality:
* Keybinds
    * Backspace - return letters to letter area
    * Enter - (try to) play word

* Field for played words and points
